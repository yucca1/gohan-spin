# 要求内容

## 概要

アプリ画面ヘッダーに表示している絵文字を、観覧車（🎡）からご飯茶碗（🍚）に変更し、あわせてブラウザタブのアイコン（favicon）も🍚に設定する。

## 背景

- READMEのトップ見出しは「🍚 gohan-spin」とご飯の絵文字を使っているが、実際のアプリ画面ヘッダーは「🎡 gohan-spin」と観覧車の絵文字になっており、アプリの「顔」が一致していない。
- 「外食するお店を決めるアプリ（ご飯）」という性質からも、観覧車よりご飯の絵文字（🍚）の方がコンセプトに合う。
- 現状、faviconが未設定でブラウザのタブにデフォルトアイコンしか出ないため、この機会に🍚で統一する。

## 実装対象の機能

### 1. ヘッダー絵文字の変更
- `index.html` のヘッダー `🎡 gohan-spin` を `🍚 gohan-spin` に変更する。

### 2. favicon の追加
- ブラウザタブに表示されるアイコン（favicon）を🍚の絵文字にする。
- SVGに絵文字を描き、data URIとして `<head>` に `<link rel="icon">` で埋め込む（追加ファイルを作らない方式）。

## 受け入れ条件

### ヘッダー絵文字の変更
- [ ] アプリ画面のヘッダーが「🍚 gohan-spin」と表示される
- [ ] 観覧車（🎡）の絵文字がコード上から無くなっている

### favicon の追加
- [ ] ブラウザのタブに🍚のアイコンが表示される
- [ ] 開発サーバー（`npm run dev`）とビルド成果物（`npm run build`）の両方で表示される

## 成功指標

- READMEトップ・アプリヘッダー・ブラウザタブの3か所すべてで🍚に統一される

## スコープ外

以下はこのフェーズでは実装しません:

- READMEの内容変更（既に🍚のため変更不要）
- ヘッダーのデザイン・レイアウト・スタイル変更
- PNG等の画像ファイルによるfavicon・PWA用アイコン（apple-touch-icon等）の整備
